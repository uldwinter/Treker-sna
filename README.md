# Treker-sna

Проект переведен на структуру React/Vite, согласованную по предоставленным скриншотам.

## Что сделано

- Создана директория `guidelines/` с `Guidelines.md`.
- Добавлен каркас `src/app/components/` с доменными подпапками (`admin`, `student`, `curator`, `ui`, и т.д.).
- Добавлены файлы слоев `layouts` и основные экраны (`LoginScreen`, `MainScreen`, `ReportsScreen`, и другие).
- Добавлены `src/styles/*.css`.
- Добавлены базовые конфиги: `package.json`, `postcss.config.mjs`, `vite.config.ts`, `ATTRIBUTIONS.md`.

## Запуск (после добавления зависимостей)

```bash
npm install
npm run dev
```
